# Epochs: Data Science Bootcamp — Submissions

Submissions for **Epochs: Data Science Bootcamp**, an 11-day program (July 20–30, 8:00–9:30 PM, online) run by μLearn Foundation's Data Science Interest Group in collaboration with SupeAI. The bootcamp covers a roadmap from SQL and data fundamentals through EDA, visualization, supervised and unsupervised ML, model deployment/MLOps, and finishes with GenAI, LLMs, and RAG.

Each day's task lives in its own folder, self-contained with its own notebook and README.
